class basecar.BaseCar[Quellcode]¶

Basisklasse für ein ferngesteuertes Fahrzeug, das Vorder- und Hinterräder steuern kann. Die Konfiguration wird aus einer JSON-Datei geladen, mit Fallback auf Standardwerte.

Attributes
direction

Gibt die aktuelle Fahrtrichtung des Fahrzeugs zurück: 1 = Vorwärts, -1 = Rückwärts, 0 = Stillstand.

speed

Gibt die aktuelle Geschwindigkeit des Fahrzeugs zurück.

steering_angle

Gibt den aktuellen Lenkwinkel des Fahrzeugs zurück.
